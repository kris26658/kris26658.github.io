About this Site

This website was made for the final project of my high school programming class. In an effort to learn HTML, the class has been making websites using a combination of the text editor VScode and the website hosting service Github. We've also been using CSS to style the website.

About Test Page

The Test Page is made for testing HTML code for class. It will change based on the assignement we are completing at the time. If you navigate to the test page, please be warned that it does not line to the same style sheet as the rest of the site, nor does it have a way to navigate between pages. Hit the back arrow in your browser to navigate away from it and back to the rest of the site.

About Get to School

Get to School is an unfinished project made for the same class. To learn programming concepts, we were taught how to use RPG Maker MV, and followed challenge pages from out teacher to create some of its features. Some examples of challenges we completed include making doors open and close when interacted with, blocking off areas until you get certain items, and creating a dungeon with a boss enemy.

Play through it on the Get to School page.
